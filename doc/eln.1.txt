ELN(1)
======
:doctype: manpage

NAME
----
eln - Electronic Lab Notebook

SYNOPSIS
--------
*eln* ['NOTEBOOK']

DESCRIPTION
-----------
The *eln*(1) command starts the ELN Electronic Lab Notebook editor.
Without arguments it presents a splash screen from which a notebook
can be selected. With argument, it loads up the specified notebook.

OPTIONS
-------
None.

BUGS
----
ELN is fairly mature and used daily in our lab. ELN has never caused
anyone to lose any data or notes. However, there may be bugs. Please
contact the author if one bites you.
